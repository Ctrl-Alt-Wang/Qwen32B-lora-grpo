## 分析与讨论

### 为什么训练 reward 涨了 +3.3% 但评测只涨 +1.4 分？

1. **训练 reward 与评测指标对齐度有限。** 训练奖励函数为 `0.5×格式分 + 0.5×RM分`，主要奖励 PICO 结构完整性，对证据引用质量的直接信号不足。
2. **r=16 主要提升了内容整合能力。** 证据回答一致性（+7.1）是最大受益项，说明更大的 rank 让模型更好地将检索结果融入结构化回答，但不改变检索本身的质量。
3. **收益递减趋势明确。** v1→v3 评测增益远大于 v3→v4，单纯扩 LoRA capacity 已触及瓶颈。

### 下一步方向建议

- **改进奖励函数**：加入引用质量奖励（引用编号与结论对齐度），直接优化评测最弱项
- **数据增强**：血液系统（59.4）、神经内科（60.9）等弱势科室补充训练样本
- **全参数 SFT 热启动**：用更优质的 SFT 基座再做 GRPO，比扩 rank 更有效
